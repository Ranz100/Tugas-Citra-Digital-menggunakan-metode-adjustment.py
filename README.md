# Tugas-Citra-Digital-menggunakan-metode-adjustment.py
Proyek ini adalah tugas manipulasi citra digital menggunakan metode bebas. Saya menggunakan kombinasi teknik CLAHE (Color Correction), Sharpen Filter, dan Brightness/Contrast Adjustment untuk menghasilkan gambar yang lebih jelas, berwarna, dan tetap terlihat seperti foto masa kini.
